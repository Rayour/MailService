# Django module

## Описание:

Проект для выполнения домашних заданий модуля по djando

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Rayour/DjangoProject.git
```
2. Установите зависимости:
```
poetry instal 
```
## Использование:

 * Загрузка групп пользователей:
```python
python manage.py add_groups
```
 * Запуск рассылки через консоль:
   Запуск рассылки через консоль не проверяет права пользователя на рассылку и блокировку пользователя так как доступ к консоли может быть только у ответственного лица. Команда является "обходным путем" запуска рассылки в непредвиденных ситуациях.
```python
python manage.py start_newsletter {newsletter_id} {user_id}
```
